Assignments

Assignments will be posted on Blackboard. Please submit your assignment using Blackboard before 11:59 pm on the due date. If possible, please submit a single, all-in-one file (PDF or MS Word) that contains summary, codes, and results (e.g., plots and tables); also include your name inside the document (not only as a file name), so that the TA can tell whose submission it is when grading a print-out.
